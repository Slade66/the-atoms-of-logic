# 创建 Jenkins Pipeline Job

## 一、准备凭证

- 先把拉取代码和 SSH 服务器要用到的凭证存到 Jenkins 全局凭证里。

    1. `Manage Jenkins`
    2. `Credentials`
    3. `System`
    4. `Global credentials`
    5. `Add Credentials`

## 二、创建任务

- **新建入口** 在 Jenkins 首页点击 `New Item`。

- **任务名称** 将任务名称设置为 `bifrost-deploy`。

- **任务类型** 选择 `Pipeline` 类型，然后点击 `OK`。

## 三、配置 Pipeline

- **配置入口** 进入任务配置页面后，滚动到最下面的 `Pipeline` 区域。

- **SCM 配置** 将 Pipeline 定义方式设置为从 Git 仓库读取 `Jenkinsfile`。

    | 字段 | 值 |
    | --- | --- |
    | `Definition` | `Pipeline script from SCM` |
    | `SCM` | `Git` |
    | `Repository URL` | `ssh://git@10.30.60.149:2222/nantian/bifrost.git` |
    | `Credentials` | `gitea-ssh-key` |
    | `Branch Specifier` | `*/main` |
    | `Script Path` | `Jenkinsfile` |

- **保存配置** 配置完成后点击 `Save`。
