# Linux 如何在后台运行命令？

- **问题：** 前台进程占住终端了，没法继续输入命令。

### 方法一：临时挂起 + 放到后台（命令已经在跑）

- **暂停当前任务：** 按 `Ctrl + Z`，会把当前任务"暂停"（不是结束）。

- **放到后台继续：** 输入 `bg`，把它切到后台继续运行。

### 方法二：启动时直接后台运行

- **语法：** 在命令末尾加 `&`，表示让程序在后台执行。

    ```bash
    your_command &
    # 例如
    python app.py &
    ```

### 方法三：关闭终端也继续运行（防止退出终端被杀）

- **用 `nohup`：** 希望退出终端后程序还继续跑，用 `nohup` 启动。

    ```bash
    nohup your_command &
    # 例如
    nohup python app.py &
    ```

- **输出重定向：** 默认会写到当前目录的 `nohup.out`。

### 把后台任务切到前台

- **查看后台任务：** `jobs` 列出当前 shell 的后台任务及其编号。

- **切回前台：** `fg %1`，其中 `%1` 是任务编号（用 `jobs` 查看）。
