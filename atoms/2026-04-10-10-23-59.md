# 把 Kilo 接入 Zed

1. 打开 Agent Panel
2. 右上角 + 号 New Thread -> Add More Agents
3. 搜索 Kilo 并下载
4. 在命令行打开 Kilo，并登录 Kilo Gateway
5. 打开 settings.json
6. 添加：
    ```json
    "agent_servers": {
        "kilo": {
        "type": "custom",
        "command": "kilo",
        "args": ["acp"],
        },
    },
    ```
