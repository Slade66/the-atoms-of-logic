# 如何在 Linux 上安装 MariaDB

1. **安装数据库服务器和客户端工具：** `yum install -y mariadb-server mariadb`

2. **启动服务：**

    ```bash
    systemctl start mariadb
    systemctl enable mariadb
    ```

3. **设置 root 密码：** `mysqladmin -u root password '123456'`

4. **开启 root 远程访问：**
   
   1. **登录数据库：** `mysql -uroot -p123456`
   
   2. **执行：**

        ```sql
        GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY '123456';
            FLUSH PRIVILEGES;
        ```

5. **监听所有 IP：**

    1. 编辑配置：`vi /etc/my.cnf`
    
    2. 把 `bind-address=127.0.0.1` 注释掉，默认监听 `0.0.0.0`

6. **放行防火墙：**

    ```bash
    firewall-cmd --permanent --add-port=3306/tcp
    firewall-cmd --reload
    ```

7. **重启服务：**

    ```bash
    systemctl restart mariadb
    ```

8. **验证：**

    ```bash
    mysql -uroot -p123456 -h[IP_ADDRESS]
    ```
