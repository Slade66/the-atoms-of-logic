# LocalSend：Android 和 Windows 之间的局域网传输工具

### 它是什么？

- **核心定位** LocalSend 是一个快速、开源、免费、支持 Android 和 Windows 的局域网文件传输软件。

- **使用场景** 它适合在自己的手机和电脑之间传照片、视频和文件，可以理解为 “Android 和 Windows 之间的 AirDrop”。

- **传输条件** 手机和电脑连接同一个 Wi-Fi 后，同时打开 LocalSend，就可以在局域网内互相发现并传输文件。

- **操作方式** 发送端选择文件，点选接收端设备名，接收端确认后即可开始传输。

- **传输安全** 它不需要登录账号，也不依赖云盘或第三方服务器。数据只在本地 Wi-Fi 网络内传输，并使用 HTTPS 加密。

### 安全风险

- **中间人风险** 1.16.1 及以下版本存在严重中间人攻击风险，同一局域网内的攻击者可能伪装设备、拦截或篡改文件；这个问题已在 1.17.0 修复。

- **代码执行风险** 1.17.0 之前还存在远程代码执行风险，恶意设备可能通过文件传输请求把文件写到任意位置，Windows 上甚至可能借启动目录执行代码；这个问题也已在 1.17.0 修复。

- **安全用法**
    
    1. 使用最新版，只在可信局域网里使用 LocalSend。
    
    2. 不要开启 Quick Save / 自动保存，尤其是在公共网络或多人局域网里。
    
    3. 传完文件后退出 LocalSend，避免它长时间在局域网暴露接收服务。

### 怎么使用？

- **安装** Android 手机去 Google Play 下载 LocalSend，Windows 去 LocalSend 官网下载页下载 LocalSend；两台设备都打开 LocalSend，并连接同一个 Wi-Fi。

- **Android 发到 Windows** 从 Android 选择内容并发送到 Windows，Windows 端确认接收。

    1. Windows 打开 LocalSend，停留在主界面即可。
    2. Android 打开 LocalSend，点击底部或侧边的 Send / 发送。
    3. 按需要选择发送内容：File / 文件用于发送单个或多个文件，Media / 图片视频用于发送照片和视频，Text / 文本用于发送一段文字，Folder / 文件夹在版本支持时可以发送整个文件夹。
    4. 选好后，下方会出现附近设备，比如你的 Windows 电脑名。
    5. 点击 Windows 电脑名。
    6. Windows 端弹出接收提示后，点击 Accept / 接收。
    7. 文件一般会保存到 Windows 的 下载 / Downloads 文件夹，默认保存位置可以在设置中更改。
