# UFW

## 一、概念

- **基本定义** UFW 全称是 Uncomplicated Firewall，是 Linux 系统上的简易防火墙管理工具，也是 Ubuntu 等 Debian 系发行版常用、推荐的防火墙配置工具。

- **适用范围** Ubuntu / Debian 通常默认预装 UFW；RHEL / CentOS / Fedora 官方默认使用 firewalld。

- **底层本质** UFW 本质上是 iptables / nftables 的前端封装，用更直观的命令管理 Linux 防火墙规则。

- **核心价值** UFW 的目标是降低防火墙配置复杂度，避免直接编写复杂的 iptables / nftables 规则。

## 二、常用命令

- **状态查看** 查看当前 UFW 状态和已有规则。

    ```bash
    sudo ufw status
    ```

- **端口放行** 按端口和协议允许访问。

    ```bash
    sudo ufw allow 22/tcp
    ```

- **端口拒绝** 按端口和协议拒绝访问。

    ```bash
    sudo ufw deny 3306/tcp
    ```

- **服务开关** 启用或禁用 UFW。

    ```bash
    sudo ufw enable
    sudo ufw disable
    ```

- **服务名称** 按常见服务名称放行规则。

    ```bash
    sudo ufw allow ssh
    sudo ufw allow http
    sudo ufw allow https
    ```

- **来源限制** 只允许指定来源 IP 访问。

    ```bash
    sudo ufw allow from 192.168.1.100
    ```

- **删除规则** 删除已有放行规则。

    ```bash
    sudo ufw delete allow 80/tcp
    ```

- **重置规则** 清空所有 UFW 规则并恢复初始状态。

    ```bash
    sudo ufw reset
    ```

## 三、默认策略

- **入站策略** UFW 通常默认拒绝所有入站流量。

- **出站策略** UFW 通常默认允许所有出站流量。
