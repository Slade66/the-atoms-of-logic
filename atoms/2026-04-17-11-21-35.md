# lsblk 列出块设备

- **作用**：以树状结构列出系统中所有的块设备（磁盘、分区、LVM、loop）。

- **只读命令**，不修改磁盘，可放心执行，无需 root。

- **查看设备树：** 直接 `lsblk` 即可。
    
    -
        ```bash
        $ lsblk
        NAME        MAJ:MIN RM  SIZE RO TYPE MOUNTPOINT
        sda           8:0    0  200G  0 disk
        ├─sda1        8:1    0    1G  0 part /boot
        └─sda2        8:2    0  199G  0 part
        └─ao-root 252:0    0  199G  0 lvm  /
        ```
    
    - **TYPE**：设备类型。`disk`（整盘）、`part`（分区）、`lvm`（逻辑卷）、`loop`（回环）。
    
    - **RM**：是否可移除。`1` 表示可移动设备（如 U 盘），`0` 表示固定磁盘。
    
    - **SIZE**：设备容量。
    
    - **RO**：是否只读。`1` = 只读。
    
    - **MOUNTPOINT**：挂载点。为空表示未挂载。

- **常用参数**
    
    - **`-f`**：显示文件系统信息——`FSTYPE`（类型）、`LABEL`（卷标）、`UUID`。
    
    - **`-p`**：显示设备的完整路径（如 `/dev/sda1`），便于直接复制使用。
    
    - **`-o <列名>`**：自定义输出列，例如 `lsblk -o NAME,SIZE,FSTYPE,MOUNTPOINT,UUID`。
