# VMware VDDK SDK 安装与验证

### 一、目标

- **核心用途** VDDK 是开发“访问 VMware 虚拟机磁盘内容”的应用程序所需的基础 SDK。

- **验证目标** 在 Linux 环境下安装 VMware Virtual Disk Development Kit（VDDK）`9.0.1`，并通过一个最小 C 程序验证 `libvixDiskLib.so` 能否正常初始化和退出。

### 二、解压安装包

- **解压命令** 先把 VDDK 安装包解压到临时目录，方便检查内部结构。

    ```bash
    mkdir -p /tmp/vddk-extract
    tar xzf /root/VMware-vix-disklib-9.0.1.0.24964809.x86_64.tar.gz -C /tmp/vddk-extract
    ```

- **解压产物** 解压后的关键目录是 `include/` 和 `lib64/`，其中 `include/` 包含 `vixDiskLib.h` 等头文件，`lib64/` 包含 `libvixDiskLib.so` 及其依赖项。

### 三、部署到固定路径

- **部署命令** 将解压后的 SDK 复制到 `/usr/lib/vmware-vix-disklib`，便于后续编译和运行时稳定引用。

    ```bash
    sudo cp -r /tmp/vddk-extract/vmware-vix-disklib-distrib /usr/lib/vmware-vix-disklib
    ```

- **部署验证** 检查目标目录是否包含 VDDK 的标准子目录。

    ```bash
    ls /usr/lib/vmware-vix-disklib/
    ```

    ```text
    bin64  doc  FILES  include  lib32  lib64
    ```

### 四、编写最小验证程序

- **验证思路** 编写 `test_vddk_init.c`，只调用 `VixDiskLib_InitEx` 和 `VixDiskLib_Exit`；前者用于初始化 VDDK 库，后者用于释放资源。

    ```c
    #include <stdarg.h>
    #include <stdio.h>
    #include <vixDiskLib.h>

    static void Noop(const char *fmt, va_list args) {
        (void)fmt;
        (void)args;
    }

    int main(void) {
        VixError err = VixDiskLib_InitEx(
            9, 0,
            Noop, Noop, Noop,
            "/usr/lib/vmware-vix-disklib",
            NULL
        );

        if (err != VIX_OK) {
            printf("VixDiskLib_InitEx failed: %llu\n", (unsigned long long)err);
            return 1;
        }

        VixDiskLib_Exit();
        printf("VDDK init ok\n");
        return 0;
    }
    ```

- **路径说明** `VixDiskLib_InitEx(9, 0, ...)` 显式传入 VDDK 主版本和次版本；`libDir` 参数应传 VDDK 安装根目录 `/usr/lib/vmware-vix-disklib`，编译链接和运行时动态库搜索路径再指向其中的 `lib64/`。

### 五、编译并运行

- **编译命令** 使用 `-I` 指定头文件目录，使用 `-L` 和 `-lvixDiskLib` 指定链接库，使用 `-Wl,-rpath` 减少对 `LD_LIBRARY_PATH` 的依赖。

    ```bash
    gcc -Wall -o test_vddk_init test_vddk_init.c \
        -I/usr/lib/vmware-vix-disklib/include \
        -L/usr/lib/vmware-vix-disklib/lib64 \
        -lvixDiskLib \
        -Wl,-rpath,/usr/lib/vmware-vix-disklib/lib64
    ```

- **运行命令** 直接执行生成的验证程序。

    ```bash
    ./test_vddk_init
    ```

- **成功输出** 看到 `VDDK init ok` 表示 VDDK 初始化和退出流程都能正常执行。

    ```text
    VDDK init ok
    ```
