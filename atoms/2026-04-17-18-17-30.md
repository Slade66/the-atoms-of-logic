# SQL 修改表结构（ALTER TABLE）

- **用途：** 使用 `ALTER TABLE` 对已存在的表新增、修改或删除列，不必重建表。

- **示例：一次新增多列，并指定插入位置**

    ```sql
    ALTER TABLE `order_main`
      ADD COLUMN `points_deduct` INT NOT NULL DEFAULT 0 COMMENT '本单使用的积分数' AFTER `total_amount`,
      ADD COLUMN `reward_points` INT NOT NULL DEFAULT 0 COMMENT '本单支付后赠送的积分数' AFTER `points_deduct`;
    ```

- **要点：**

    - **`ADD COLUMN`：** 新增列，多个列用逗号分隔，一条语句内完成。

    - **`AFTER <列名>`：** 指定新列插入在哪一列之后（MySQL 特有），不写则默认加到末尾。

    - **`NOT NULL DEFAULT 0`：** 建议为新列设定默认值，避免已有行回填失败。

    - **`COMMENT`：** 给列加注释，便于后续维护。
