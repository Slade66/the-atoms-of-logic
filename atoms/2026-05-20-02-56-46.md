# macOS 使用 OpenConnect 连接 FortiGate VPN

## 一、安装 OpenConnect

- **安装命令** 使用 Homebrew 安装 `openconnect`。

    ```bash
    brew install openconnect
    ```

- **版本检查** 安装完成后，确认 `openconnect` 可以正常使用。

    ```bash
    openconnect --version
    ```

## 二、连接 VPN

- **连接步骤** 按顺序完成下面的操作。

    1. 执行连接命令：

        ```bash
        sudo openconnect --protocol=fortinet 183.62.240.234:9288
        ```

    2. 输入 `yes` 确认连接。

    3. 输入 macOS 密码。

    4. 输入 VPN 账号和密码。
