# Appsmith 根据主键或外键显示可读性名称

1. 在左侧菜单新建查询（Queries），查询名设为 `GetReviewers`，SQL 语句写 `SELECT no, name FROM users;`。

2. 在页面控件中，将 `Default value` 填为以下 JS 表达式：
    ```js
    {{ GetReviewers.data.find(r => r.no == sourceData.reviewer_no)?.name }}
    ```
    - 其中 `r` 代表数据库返回的每一行记录。
