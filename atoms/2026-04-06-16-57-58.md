# 如何注册 Claude 账号？

1. 准备一个 Proton 邮箱、干净的美国静态住宅 IP、指纹浏览器。

2. 去 https://www.textverified.com/ 注册账号，充值 2.5$（最少也要充这么多）。

3. Verifications -> New SMS Verification -> 搜索 Claude -> 点击之后租用一个手机号。

4. 复制电话号码，填入 Claude 注册页面，如果提示电话号码无效，就换一个（不收费）。

5. 点击发送验证码，把收到的验证码填入 Claude 注册页面。
