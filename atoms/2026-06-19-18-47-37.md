# Cisco Packet Tracer 如何下载？

- **说明：** 官方下载只有一个正规入口：思科网络学院（Cisco Networking Academy，netacad.com）。下载链接不在公开页面上，而是藏在课程资源里，必须先注册账号、报名一门免费课程才能拿到。

- **步骤：**
    
    1. 打开[思科网络学院](https://www.netacad.com/courses/getting-started-cisco-packet-tracer)
    
    2. 注册一个账号。

    3. 报名那门免费的「Getting Started with Cisco Packet Tracer」入门课程。

    4. 进课程后，在「1.0. Install Cisco Packet Tracer」下的「1.0.3 Download Cisco Packet Tracer」里就能找到[下载链接](https://www.netacad.com/resources/lab-downloads)。
