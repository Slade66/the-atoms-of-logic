# 如何在 CentOS 7 中配置 IP 地址？

### 1. 配置文件路径

- 在 CentOS 7 中，所有的网络持久化配置都存储在 `/etc/sysconfig/network-scripts/` 目录下以 `ifcfg-` 开头的文件中。

> [!TIP]
> 可通过 `ip addr` 命令确认你本机的网卡确切名称（如 `ens192` 或 `eth0`）。

### 2. 方案一：配置静态 IP（Static）

- 适用于服务器环境，确保 IP 地址固定，方便后续服务连接。

- **核心配置示例：**

    ```bash
    TYPE=Ethernet
    BOOTPROTO=static    # 关键：设置为静态模式
    NAME=ens192
    DEVICE=ens192
    ONBOOT=yes          # 关键：设置为开机自动启用网卡

    IPADDR=10.30.65.212 # 设定的 IP 地址
    PREFIX=24           # 子网掩码长度（24 位等同于 255.255.255.0）
    GATEWAY=10.30.65.1  # 默认网关（必须有，否则无法跨网段通信）
    DNS1=8.8.8.8        # DNS 服务器（必须有，否则无法解析域名）
    ```

---

### 3. 方案二：配置动态获取（DHCP）

- 适用于具备 DHCP 服务器的自动化环境，管理简单。

- **核心配置示例：**

    ```bash
    TYPE=Ethernet
    BOOTPROTO=dhcp      # 关键：设置为动态获取模式
    NAME=ens33
    DEVICE=ens33
    ONBOOT=yes
    ```
- **注意**：在 DHCP 模式下，无需填写 `IPADDR`、`GATEWAY` 和 `DNS` 字段，这些信息会由 DHCP 服务器在系统联网时自动下发。

---

### 4. 如何使配置生效？

- 修改完成后，直接重启网络服务即可：

    ```bash
    # 重启网卡服务
    systemctl restart network

    # 验证 IP 是否生效
    ip addr
    ```
