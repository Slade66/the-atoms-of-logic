# 如何验证 CBT 开启成功？

### 途径 1：查底层 Datastore 文件

- **CBT 真正生效的唯一标志：** 是否生成了对应磁盘的 -ctk.vmdk 文件。

- **验证步骤：**
    
    1. 登录 ESXi Web 客户端。
    
    2. 导航至 Virtual Machines，并在列表中点击目标虚拟机。
    3. 展开 Hard disk 1，在 Backing 一栏中点击其所在的存储卷（例如 `datastore1`）。
    4. 点击 Datastore browser（数据存储浏览器），进入该虚拟机的目录。
    5. 确认是否存在 `[虚拟机名]-ctk.vmdk` 文件。如果有，则说明 CBT 功能已生效。

### 途径 2：查看 vSphere 图形界面高级设置

1. 在 ESXi Web 客户端中选中目标虚拟机。

2. 右键选择 编辑设置 (Edit Settings) -> 切换至 虚拟机选项 (VM Options) 选项卡 -> 展开 高级 (Advanced) -> 点击右侧“配置参数”（Configuration Parameters）旁的 编辑配置 (Edit Configuration)。

3. 在弹出的参数列表中查找 `ctkEnabled`。

4. 预期结果：全局参数 `ctkEnabled` 的值将显示为 `true`，同时各虚拟磁盘对应的参数（如 `scsi0:0.ctkEnabled` 等）的值也均会变为 `true`。
