# 不用 IntelliJ IDEA 如何创建 Spring Boot 项目

1. 访问 [Spring Initializr](https://start.spring.io/)。

2. 配置项目基本信息：依赖管理工具（Maven）、开发语言（Java）、Spring Boot 版本、项目信息（Group、Artifact）、打包方式（Jar 或 War）、配置文件格式（Properties 或 YAML）以及 Java 版本（21 或 17）。

3. 点击右上角的“添加依赖”按钮，按需添加项目依赖，例如 Spring Web、Lombok、MySQL Driver、MyBatis Framework。

4. 下载、解压。
