# MCP 协议有什么用？

**MCP 是“连接 LLM 和外部世界”的协议：** Model Context Protocol（MCP）是一个开放的协议，核心目标是实现 LLM 应用与外部数据源和工具之间的集成。让 AI 能在应用服务器上执行操作、获取数据。

AI 想要最大化发挥作用，需要连接外部的数据源，操作外部的系统。而 LLM 本身只会生成文本，不能直接访问外部数据，需要调用工具。这遇到如下的问题：

[[MCP 解决了「能力发现」问题]]
