# go doc：终端里的即时 API 说明书

- `go doc` 是 Go 语言自带的文档查看工具，通过它可以在终端快速调阅包、类型、函数或方法的定义与说明。

### 1. 核心价值：AI 开发的“避坑”器

- **应用场景**：在让 AI 调用任何（尤其是第三方库）API 之前，强制要求其先行运行 `go doc` 检索接口定义。

    - **消除幻觉**：斩断 AI 凭“大概记忆”臆测参数类型、属性名或返回值结构的根源。
    
    - **对齐版本**：确保生成的代码符合当前开发环境的库版本，规避因 API 变更导致的“过期代码”报错。

    - **降低成本**：极大减少因调用错误引发的低效 Debug 与返工。

### 2. 核心用法与常用命令

- **查看包概览**：
  
    ```bash
    go doc fmt          # 查看 fmt 标准库简介
    go doc net/http     # 查看 http 包简介
    ```

- **查看具体标识符**：
  
    ```bash
    go doc fmt.Println      # 查看具体函数的参数与说明
    ```

- **查看类型的完整方法集**：
  
    ```bash
    # 查看包内某个类型的定义并罗列该类型所有可导出的方法
    go doc bytes.Buffer
    ```
