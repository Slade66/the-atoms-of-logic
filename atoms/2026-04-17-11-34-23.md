# 设备文件是什么？

- **普通文件：** 存的是数据（存在磁盘上）。

- **设备文件：** 是入口 / 句柄，本身不存数据，只是内核驱动的一个代理。

- **核心概念：** Linux 把"硬件设备"抽象成文件，统一放在 /dev/ 目录下。应用程序读写硬件，就像读写普通文件一样 —— 这就是 Unix 哲学："一切皆文件"。

- **逻辑图示：**
    ```text
    用户程序                 内核                  硬件
    │                      │                     │
    │  read("/dev/sda")    │                     │
    ├─────────────────────►│                     │
    │                      │  驱动程序           │
    │                      ├────────────────────►│ 磁盘
    │                      │                     │
    │◄─────── 数据 ────────┤◄────────────────────┤
    ```
