# 设备文件是什么？

- **普通文件：** 存的是数据（存在磁盘上）。

- **设备文件：** 设备文件是 Linux 中的一种特殊文件，位于 `/dev` 目录下，用来作为用户访问硬件设备的接口。通过对设备文件进行读写操作，实际上是在与内核中的设备驱动交互，从而操作真实设备。

- **核心概念：Everything is a file（一切皆文件）** Linux 把"硬件设备"抽象成文件，统一放在 `/dev/` 目录下。应用程序读写硬件，就像读写普通文件一样统一接口，所有东西都可以用：`open`，`read`，`write`，`close`。

    - **比如：** `echo "hello" > /dev/tty` 实际是在往终端设备写数据。

- **逻辑图示：**
    ```text
    用户程序                 内核                  硬件
    │                      │                     │
    │  read("/dev/sda")    │                     │
    ├─────────────────────►│                     │
    │                      │  驱动程序           │
    │                      ├────────────────────►│ 磁盘
    │                      │                     │
    │◄─────── 数据 ────────┤◄────────────────────┤
    ```
