# Claude 封号的两个关键因素

## 一、IP 纯净度

- **核心判断：** 平台会检查访问 IP 是否像普通用户的家庭宽带，而不是机房或高风险网段。

- **ASN 属性：** IP 所属的自治系统编号（ASN）会暴露网络来源。

    - `Data Center` 或 `Hosting` 通常代表机房 IP，例如阿里云、AWS、搬瓦工等 VPS。
    - `Residential` 或 `ISP` 通常代表住宅宽带 IP，例如 Comcast、AT&T 等家庭网络运营商。

- **污染风险：** 即使 IP 本身是家宽，如果同一网段被其他用户滥用过，也可能被平台标记为高风险。

- **检测方法：** 可用多个网站交叉检查 IP 风险。

    - [ipinfo.io](https://ipinfo.io/)：重点看 `type` 是否为 `isp`，以及 `company` 是否为真实宽带运营商。

    - [scamalytics.com](https://scamalytics.com/)：重点看 `Fraud Risk` 是否为 `Low Risk`。

    - [ippure.com](https://ippure.com/) 和 [ping0.cc](https://ping0.cc/)：辅助判断 IP 纯净度。

## 二、浏览器指纹

- **关联风险：** 如果多个账号共享同一个 IP，同时浏览器指纹又高度一致，平台可能把这些账号关联为同一批量操作者。

- **伪造方式：** 指纹浏览器会伪造或隔离关键环境参数，让每个账号看起来像来自不同设备。

    - 给 Canvas/WebGL 渲染结果注入细微噪声。
    - 修改 `navigator` 返回的硬件信息。
    - 伪造字体、语言、时区和地理位置。
    - 隔离每个 Profile 的 Cookies、缓存和本地存储。

- **一致性要求：** 浏览器使用美国 IP 时，不应返回中国时区。
