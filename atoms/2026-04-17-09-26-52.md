# VMware ESXi HotAdd

- **作用：**  VMware 的一项能力，允许在虚拟机运行时动态挂载 SCSI 磁盘。

- **场景：** 在备份/迁移场景中，一个运行在同一 vSphere 环境中的备份代理 VM（Backup Proxy VM），通过 HotAdd 将目标 VM 的虚拟磁盘（VMDK）热挂载到自身，然后直接从数据存储读取磁盘数据，绕过网络，实现高效数据传输。

- **限制：**
    1. Proxy VM 必须安装最新版 VMware Tools
    2. 目标 VM 磁盘类型必须是 SCSI，不支持 IDE
    3. Proxy VM 需要能访问源 VM 所在的 datastore

- **在 ESXi 网页控制台用 HotAdd 挂载磁盘的步骤：**
    1. 找到 Proxy VM → 编辑设置
    2. 添加硬盘 → 现有硬盘，选一个现有 VMDK
    3. 保存后进入 Proxy VM 检查是否有新磁盘出现（`lsblk`）
    4. 如果成功出现 → 说明 ESXi 环境支持 HotAdd
