# Codex 怎么配置 1M 上下文？

1. 打开 `~/.codex/config.toml`

2. 添加以下配置：

    ```toml
    model = "gpt-5.4"
    model_context_window = 1000000
    model_auto_compact_token_limit = 900000
    ```

- **注意：** GPT-5.5 在 Codex 中的上下文窗口被限制为 400K。即使模型本身支持 1M，你在 `config.toml` 里配置了 1M，实际也会静默回落到 258K。

   - **原因：** 这是因为 GPT-5.5 一旦输入超过 272K token，整个 session 的 input 价格会变为 2 倍，output 价格会变为 1.5 倍。换句话说，1M 上下文会显著抬高 OpenAI 的成本；而 Codex 订阅是固定月费，在这个定价下，OpenAI 无法承担所有用户都使用 1M 上下文的成本。
