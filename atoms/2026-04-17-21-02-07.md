# docker save

- **作用：** 把 Docker 镜像导出为一个 `tar` 文件，方便备份、传输或离线部署。

- **基本用法：**

    - **语法：**

        ```bash
        docker save -o <文件名.tar> <镜像名:标签>
        ```

    - **示例：** 把 `nginx:latest` 镜像保存为 `nginx.tar`。

        ```bash
        docker save -o nginx.tar nginx:latest
        ```

- **导出多个镜像：** 一个 tar 文件里可以包含多个镜像。

    ```bash
    docker save -o images.tar nginx:latest redis:7 alpine:3.18
    ```

- **输出到 stdout 配合管道符：** 把镜像流式输出到 `stdout`，可以配合管道符传输到其它地方。

    ```bash
    docker save nginx:latest | gzip > nginx.tar.gz
    ```

### 注意事项

- **docker save 保存的是镜像：**

    - 如果你想保存容器当前状态，用：`docker export <container>`
