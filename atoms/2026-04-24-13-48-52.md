# 如何配置 Crush 接入 DeepSeek

1. 运行 `crush dirs` 查看本机配置文件和数据目录的实际路径。

2. 创建配置目录并打开配置文件：

    ```powershell
    mkdir -Force "A:\Users\liyuz\.config\crush"
    zed "A:\Users\liyuz\.config\crush\crush.json"
    ```

3. 将以下配置内容粘贴进去保存即可：

```json
{
  "$schema": "https://charm.land/crush.json",
  "providers": {
    "deepseek": {
      "type": "openai-compat",
      "base_url": "https://api.deepseek.com/v1",
      "api_key": "$DEEPSEEK_API_KEY",
      "models": [
        {
          "id": "deepseek-v4-flash",
          "name": "DeepSeek V4 Flash",
          "cost_per_1m_in": 0.138,
          "cost_per_1m_in_cached": 0.028,
          "cost_per_1m_out": 0.276,
          "context_window": 1048576,
          "default_max_tokens": 384000,
          "supports_tools": true
        },
        {
          "id": "deepseek-v4-pro",
          "name": "DeepSeek V4 Pro",
          "cost_per_1m_in": 1.655,
          "cost_per_1m_in_cached": 0.138,
          "cost_per_1m_out": 3.31,
          "context_window": 1048576,
          "default_max_tokens": 384000,
          "supports_tools": true,
          "can_reason": true
        }
      ]
    }
  }
}
```
