# IMC：抹平 CPU 差异

- **问题** 集群内主机 CPU 代际不同时，VM 在高代 CPU 上启动后使用了高级指令集，迁移到低代 CPU 会因指令集不支持而崩溃。

- **原理** IMC（Intelligent Machine Check / Compatibility）采用“向下取平”——设定基准指令集（Baseline），所有主机对 VM 只展示基准线以内的功能，消除迁移时的“水土不服”。

- **注意** 开启 IMC 通常需要 VM 处于关机状态才能生效，因为要修改 VM 看到的硬件“面具”。
