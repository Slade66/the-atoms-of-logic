
# Project - Bifrost

### 项目背景

- 随着企业数据中心基础设施的国产化替代趋势的加速，将核心业务从VMware vSphere（ESXi）虚拟化平台迁移至基于 KVM 深度定制的华为 FusionCompute 架构，已成为众多企业的核心战略。

- 在这一异构环境的转换过程中，传统的冷迁移方案需要较长的业务停机窗口，无法满足金融、运营商等关键业务领域对高可用性和连续性的严苛要求。

- 因此，开发一套能够实现“在线迁移”（即在源端虚拟机持续提供服务的同时进行数据同步，仅在最终割接时产生分钟级中断）的自动化平台显得至关重要。它不仅能大幅降低传统手工迁移工程所带来的不可控运维风险与人力成本，还能将长达数小时甚至数天的迁移过程对业务的影响压缩至最小。

### 项目目标

- 在业务几乎零感知、确保数据字节级绝对一致性的情况下，将 VMware ESXi 上的虚拟机（包括 Windows 和 Linux 操作系统）迁移到华为 FusionCompute。

### 项目命名寓意

- **中文名：** 彩虹桥。

- **寓意：** 北欧神话中连接神界与人界的通道。

- **命名背景：** 华为官方的迁移工具叫 Rainbow（彩虹），我们用 Bifrost 是致敬。
