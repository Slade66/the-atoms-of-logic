# CMDB

- **一句话解释：** CMDB（Configuration Management Database，配置管理数据库）是存储构成企业 IT 环境的各种硬件和软件资产（通常称为配置项，CIs）信息，并展现资产间依赖关系的系统。

- **形式多样，丰俭由人：** CMDB 可以很简单（一张 Excel 表格），也可以很复杂（企业级 CMDB 能够通过扫描网络自动发现并填充资产信息，映射资产间的依赖关系，并与其它系统集成）。

- **配置项：**
    - 在 CMDB 中，这些被管理的组件被称为配置项（CIs, Configuration Items）。
    - 可以是硬件（如服务器、交换机、路由器）、软件（数据库）、虚拟机。
