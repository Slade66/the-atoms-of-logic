# 安装 docker compose

- Docker Compose v2 已集成到 Docker CLI 的插件系统中，取代了旧的独立程序。现在是 `docker compose`，不是 `docker-compose`。

- **插件安装命令：**

    ```bash
    sudo apt update
    sudo apt-get install docker-compose-plugin
    ```

- **验证（显示版本号）：**

    ```bash
    docker compose version
    ```
