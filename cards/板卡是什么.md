# 板卡是什么？

### 🧠 盒式设备（固定端口）

- **一句话定义：** 所有接口直接焊死在主板上，出厂时端口数量和类型就永久固定，没有任何板卡可插。

- **外观：** 体积不大，通常是扁平的长方形盒子，占 1U 或 2U 机架高度。就像你家里的路由器，或带 24 个网口的交换机铁盒子——面板上有几个洞、是千兆还是万兆，全都焊死在主板上了。

- **端口命名：** 格式简单直接，例如 `GigabitEthernet0/0/1` 到 `GigabitEthernet0/0/24`。

- **三大痛点：**

    - **无法平滑扩容** 假设公司买了 48 口的交换机，后来公司新招了人，需要 50 个网口。没有办法：只能再花大价钱买一台新的 48 口交换机，然后把两台用线连起来，既贵又麻烦。

    - **升级换代成本极高** 两年前买的全是千兆网口交换机（1G），现在业务升级需要万兆（10G）乃至 40G、100G。旧设备没法改造，只能整机淘汰，极度浪费。

    - **牵一发而动全身** 交换机上一个小小的电源坏了，或某个芯片烧了导致几个端口失效，只能把整台设备断电搬走维修。在这个过程中，插在上面的所有几十根网线全得拔下来，整个部门乃至全公司的网都断了。

### 🧠 框式设备（模块化大机框 + 板卡）

- **发明背景：** 为了彻底解决盒式设备的三大痛点，思科、华为、华三等网络大厂发明了"模块化大机框 + 板卡（Line Card）"设计。

- **这就像把一台电脑拆开来卖：**

    - **机框（Chassis）** 体积非常庞大（有时接近半个冰箱），买回来只是一个空铁柜子，内部只有"大主板（背板）"、超大电源和散热风扇，一个网口都没有——正面是一层一层空荡荡的插槽（Slot）。

    - **板卡（Line Card）** 相当于台式电脑里的独立显卡。它是一块长条形的"电路板带面板模块"，上面带有真正对外的网络接口，可以像抽屉一样插进机框的槽位里。你需要什么接口，就单独购买相应规格的板卡插进去。

- **三大痛点的逐一解法：**

    - **按需购买，平滑扩容** 今年需要 48 个千兆口，就只买一块"48 口千兆板卡"插在 Slot 0。明年需要扩容，再买一块板卡插进 Slot 1 即可，完全不需要买整台新机器。

    - **极其灵活升级** 五年后公司需要 100G 光口？旧机框不用扔，直接买一块"100G 速率板卡"插进去，不仅能升速，还能和旁边槽位里的旧千兆板卡协同工作。

    - **热插拔，永不断网** 1 号板卡坏了，不需要给机框断电。直接拔出坏的板卡，插上一块新的，其余槽位上的业务完全不受影响，网一秒钟都不掉。这正是核心机房必须使用框式设备的根本原因。

- **外观示意：**

    - ```text
      ========================================================================
      |                            超级大风扇组 (Fan Tray)                    |
      ========================================================================
      |                                                                      |
      | [Slot 0] 主控板卡 (大脑, MPU) - 不插网线, 负责整机的指挥和路由计算       |
      | [ * CPU * * * SYSTEM OK * * * ACT/STBY * * * ]                       |
      |----------------------------------------------------------------------|
      | [Slot 1] 主控板卡 (备用大脑, MPU) - 防止槽0坏掉，这是个一模一样的备胎    |
      | [ * CPU * * * SYSTEM OK * * * ACT/STBY * * * ]                       |
      |======================================================================|
      |                                                                      |
      | [Slot 2] 业务板卡 0 (Line Card 0) - "100GE 速率板卡"                  |
      | 名字: 100GE1/0/1  100GE1/0/2  100GE1/0/3  100GE1/0/4                 |
      |                                                                      |
      |----------------------------------------------------------------------|
      |                                                                      |
      | [Slot 3] 业务板卡 1 (Line Card 1) - "千兆电口板卡"                    |
      | 名字: GE1/1/1     GE1/1/2     GE1/1/3     GE1/1/4                    |
      |                                                                      |
      |----------------------------------------------------------------------|
      |                                                                      |
      | [Slot 4] 空槽位 (Empty Slot) - 盖着防尘挡板，以后需要了再买板卡插这里    |
      | [ //// 防尘挡板条，未插入任何业务板卡 /////////////////////////////// ] |
      |                                                                      |
      |----------------------------------------------------------------------|
      |                                                                      |
      | [Slot 5] 业务板卡 2 (Line Card 2) - 又是一块 "100GE 速率板卡"          |

      |         冗余电源 A (Power)        |         冗余电源 B (Power)        |

### 🧠 框式设备端口命名规则：`[速率][设备号]/[板卡槽位号]/[端口序号]`

- **拆解示例：** 以 `100GE1/1/2` 为例（华为/华三常见命名法）：

    - `100GE`：接口速率为 100Gbps（100G 以太网口）。

    - 第一个 `1`（设备/机框号）：若两台大设备堆叠组合成一台虚拟设备，第一台以 1 开头，第二台以 2 开头。单机默认为 1。

    - 第二个 `1`（板卡槽位号）：该板卡插在机框的第 1 号插槽里。

    - 最后的 `2`（端口序号）：在这块特定的板卡上，从左往右数的第 2 个物理接口。
