# Zap 简介与安装

### 🧠 Zap 是什么

Zap 是 Uber 开源的高性能 Go 语言日志库，旨在解决标准库 `log` 在性能和功能上的两大短板：无结构化支持、无日志分级。

### 🧠 三大核心优势

- **高性能：** 基于"零内存分配"理念设计，避免反射和 `interface{}` 开销，比标准库快数倍。
- **原生结构化：** 日志以键值对形式输出，天然适配 ELK / Loki / Splunk 等日志采集系统进行解析和检索。
- **细粒度分级：** 提供从 `Debug` 到 `Fatal` 的完整日志级别体系，弥补标准库 `log` 只有 `Print` 和 `Fatal` 的粗糙设计。

### 🧠 安装命令

```bash
go get -u go.uber.org/zap
```
