### 句柄是什么？

**一句话总结：** 句柄就是「用来代表并操作某个资源的凭证」。

**最形象的比喻**：

你去银行存钱：
🏦 钱 → 真实资源
🎫 存单 → 句柄
你不能直接操作“金库里的钱”，
只能拿“存单”去操作。

### Windows 的窗口句柄

在 Windows API 里，窗口句柄不是窗口本体，而是系统用来定位窗口的 ID。

### 文件句柄：

```c
FILE *fp = fopen("test.txt", "r");
```
这里的 fp 就是句柄：
它 ≠ 文件本身
它 = 系统给你的一个“访问入口”
你拿着句柄，系统才知道你在操作哪个文件。
